# **App Name**: VideoSpark

## Core Features:

- Niche Input: Allow users to input their desired video niche or topic.
- Trending Video Search: Search for currently trending videos related to the specified niche using an external API. 
- Idea Generation: Generate 5 video ideas with brief descriptions, inspired by trending content. Use a tool to combine insights from top videos and create novel suggestions.
- Detailed Idea Expansion: Upon request, provide more information on a specific idea, including script outlines, thumbnail concepts, title suggestions, and video description drafts.
- User Interface for Display: Display the ideas, descriptions, and expanded information in an easily readable and navigable format.

## Style Guidelines:

- Primary color: Vibrant purple (#A050BE) to convey creativity and inspiration.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font, will be used throughout the application.
- Use simple and clear icons for navigation and representing different video categories.
- Subtle transitions and animations to enhance user experience and guide attention.
